# The MiSTer Manuals DB - Sega SG-1000

This is a database for the MiSTer project that downloads the English manuals in .pdf form for the Sega SG-1000 to the docs folder, which can be loaded from the OSD within th core by selecting "Help".

To use it simply copy and paste the below to the bottom of your downloader.ini file (found at: /media/fat/downloader.ini )

```ini
[ajgowans/manualsdb-segasg1000]
db_url = https://raw.githubusercontent.com/ajgowans/manualsdb-segasg1000/db/db.json.zip
```
